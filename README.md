# Spices

This repo contains information I used to build a spice rack.

In order to create the spice labels, I used images from botanical books that are in the public domain. Some of these books were easily found on archive.org, but some were really hard to find. I did my best to use only images that are in the public domain, if you have any concerns about it open an issue or send me an email.

I processed each PDF using Adobe Acrobat's OCR, to make finding the right plant easier.
